git命令集：

1、git使用前配置
git --version
git config --list
git config --global user.name 提交人姓名
git config --global user.email 提交人邮箱地址

2、提交步骤
git init
git status
git add

这是说明文档。

 git remote add origin https://github.com/zhangkmsjtu/git-demo.git
简化处理
git push -u origin master

git push
